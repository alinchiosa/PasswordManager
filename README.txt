In order to be able tu run the application you need to:
- have Python 3.6.5 installed
- have PyCharm installed
- create a database schema in MySQL Workbench with the name "password_manager"
- import the database tables from the Database folder
- change the 'user' and the 'password' with your information in the DatabaseConnct.py file
- open the PasswordManager.py file in PyCharm
- make sure you have all the following packages added to the interpreter: PyNaCl, mysql-connector-python
- run the PasswrodManager.py file
